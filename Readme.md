# Shipment Transit Performance Analysis

This project analyzes courier shipment tracking data to evaluate
transit efficiency, facility usage, and delivery performance.

## Key Features
- Parses nested JSON shipment data
- Handles real-world data edge cases
- Computes transit time and facility metrics
- Evaluates delivery success rates
- Generates detailed and summary CSV reports

## Technologies Used
- Python
- Pandas
- JSON
- Jupyter Notebook

## Output
- transit_performance_detailed.csv
- transit_performance_summary.csv

This project was created for learning and portfolio purposes.
